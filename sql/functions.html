SQL Functions

A Function is a special type of predefined command set that performs some operation and returns a single value.
Some builtin functions provided by SQL are:

GROUP Function

A GROUP function in sql performs a calculation on multiple values and returns a single value.

1.SUM=
This function adds up all the selected values from a given expression.

Syntax:
sum([distinct | all] expression).



2.COUNT=
This function counts the number of rows in a given expression.

Syntax:
count([distinct | all] Expression)
OR
count(*)



3.MAX=
This function returns the maximum value from a given expression.

Syntax:
max([distinct | all] expression )



4.MIN=
This function returns the minimum value from a given expression.

Syntax:
min([distinct | all] expression)



5.AVG=
This function computes the average of given data.

Syntax:
avg([distinct | all] expression)



CHARACTER Function

CHARACTER Function accept character inputs and can return either characters or number values as output.

1.LOWER=
This function converts a string into lowercase.

Syntax:
lower(str)
OR
lcase(str)

2.UPPER=
This function converts a string into uppercase.

Syntax:
upper(str)
OR
ucase(str)



3.CONCAT=
This function concatenates two or more strings together to form one string.

Syntax:
concat(string1,string2,...,stringN)



4.LENGTH=
This function returns the length of a string in bytes.

Syntax:
length(expression)

5.SUBSTR=
This function extracts a substring from a given string.

Syntax:
substr(string,start [,length])



6.INSTR=
This function searches for given second string into the given first string.

Syntax:
instr(string,search_string)



DATE Function

DATE Function operate on values of the DATE datatype. In various scenarios instead of date, datetime (time is also involved with date) is used.
DATE format YYYY-MM-DD
DATETIME format: YYYY-MM-DD HH:MI: SS
TIMESTAMP format: YYYY-MM-DD HH:MI: SS
YEAR format YYYY or YY

1.CURDATE=
This function returns the current date.

Syntax:
curdate()
OR
current_date()

2.DATE=
This function extracts the date part of a date or datetime expression.

Syntax:
date(expression)

3.MONTH=
This function returns the month from the date passed.

Syntax:
month(date)

4.MONTHNAME=
This function returns the name of the month for a date.

Syntax:
monthname(date) 5.DAY=
This function returns the day part of a date.

Syntax:
day(date)



6.YEAR=
This function returns the year part of a date.

Syntax:
year(date)

7.DAYOFMONTH=
This function returns the day of month.

Syntax:
dayofmonth(date)

8.DAYOFWEEK=
This function returns the day of week.

Syntax:
dayofweek(date)

9.DAYOFYEAR=
This function returns the day of the year.

Syntax:
dayofyear(date)



10.NOW=
This function returns the current date and time.

Syntax:
now()
