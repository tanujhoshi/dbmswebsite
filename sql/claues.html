SQL CLAUSES

Clauses in SQL are similar to conditions in high-level languages. SQL being a query language requires a method to apply constraints on the data and for this we use Clauses.
Clauses help us to restrict and manage the data using valid constraints on the data in our database.

We have variety of clauses available in SQL which are:

WHERE Clause

Syntax:
select * from table_name where condition;



ORDER BY Clause

Syntax:
Select * from table_name order by column1,column2...;



GROUP BY Clause

Syntax:
select * from table_name group by column1,column2..;



HAVING Clause

Note:Having clause can be used only after Group By clause.

Syntax:
Select column from table_name group by column having condition;
